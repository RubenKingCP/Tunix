# Αρχική Περιγραφή του Έργου
Σκοπεύουμε να υλοποιήσουμε μια εφαρμογή αναπαραγωγής μουσικής. 

Ο χρήστης έχει την δυνατότητα να ακούει μουσική, είτε αναζητώντας συγκεκριμένο περιεχόμενο είτε επιλέγοντας έτοιμες προτάσεις από την εφαρμογή και να κατεβάζει τα τραγούδια που θέλει. Μπορεί να ακολουθεί καλλιτέχνες ώστε να λαμβάνει ενημερώσεις από αυτούς. Έχει την επιλογή να αλλάξει τις προτάσεις του συστήματος αφαιρόντας τραγούδια από τις προτάσεις όταν εμφανίζονται και επιλέγοντας τα αγαπημένα του τργούδια, τα οποία έμμεσα προσθέτει σε μια προσωπική λίστα αναπαραγωγής. Μπορεί να φτιάξει και επιπλέον τέτοιες λίστες.
Επιπλέον, ο χρήστης μπορεί να αιτηθεί να αναβαθμίσει τον λογαριασμό του σε λογαριασμό καλλιτέχνη. Ένας καλλιτέχνης μπορεί να κοινοποιεί νέα τραγούδια, να δημιουργεί επίσημα άλμπουμ, να ενημερώνει το κοινό του, να προπωλεί την μουσική του και να ελέγχει τα στατιστικά των αναρτήσεων του.
Η χρήση της εφαρμογής αυτής είναι αποκλειστικά δωρεάν. Σκοπός της είναι να μιμιθεί τις πιο διάσημες εφαρμογές αναπαραγωγής τραγουδιών, χωρίς να επιβαρύνει τον χρήστη με επιπλέον χρηματικό κόστος ή διαφημήσεις που διακόπτουν την ακρόαση. Επίσης η εφαρμογή αυτή παρέχει ασφαλές περιβάλλον για τους νεότερους χρήστες, αφού γίνεται αναφορά και αποκλεισμός χρηστών που κοινοποιούν ακατάλληλο περιεχόμενο από τους διαχειριστές. Αυτοί αφαιρούν το εν λόγω περιεχόμενο από την εφαρμογή και διαχειρίζονται τους διάφορους καλλιτέχνες, ελέγχοντας τους για όποιο ιστορικό παραβάσεων έχουν στην εφαρμογή. Έαν υπάρχει οποιδήποτε θέμα, έχουν την ευελιξία είτε να δώσουν μια παρατήρηση (ονομαζόμενη "strike") είτε να αποκλείσουν άμεσα τους παραβάτες.
