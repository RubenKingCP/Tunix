# Αρχική Περιγραφή του Έργου
Σκοπεύουμε να υλοποιήσουμε μια εφαρμογή αναπαραγωγής μουσικής. 

Ο χρήστης έχει την δυνατότητα να ακούει μουσική, είτε αναζητώντας συγκεκριμένο περιεχόμενο είτε επιλέγοντας έτοιμες προτάσεις από την εφαρμογή και να κατεβάζει τα τραγούδια που θέλει. Μπορεί να ακολουθεί καλλιτέχνες ώστε να λαμβάνει ενημερώσεις από αυτούς όπως και να προαποθηκεύει μουσική που ακόμα δεν έχει αναρτηθεί από καλλιτέχνες. Έχει την επιλογή να αλλάξει τις προτάσεις του συστήματος αφαιρώντας τραγούδια από τις προτάσεις όταν εμφανίζονται και επιλέγοντας τα αγαπημένα του τργούδια, τα οποία έμμεσα προσθέτει σε μια προσωπική λίστα αναπαραγωγής. Μπορεί να φτιάξει και επιπλέον τέτοιες λίστες.
Επιπλέον, ο χρήστης μπορεί να αιτηθεί να αναβαθμίσει τον λογαριασμό του σε λογαριασμό καλλιτέχνη. Ένας καλλιτέχνης μπορεί να κοινοποιεί νέα τραγούδια, να δημιουργεί επίσημα άλμπουμ, να ενημερώνει το κοινό του και να ελέγχει τα στατιστικά των αναρτήσεων του.
Σεβόμενη τα πνευματικά δικαιώματα των καλλιτεχνών της, η εφαρμογή μας θα είναι διαθέσιμη προς διαφήμιση τρίτων εταιρειών. Εάν κάποιος χρήστης δεν επιθυμεί να διακόπτεται η ακρόαση του από διαφημίσεις, μπορεί να αναβαθμίσει τη συνδρομή του σε Premium συνδρομή καταβάλλοντας ένα μικρό χρηματικό ποσό μηνιαία. Επιπλέον, αν ο χρήστης μετανιώσει την αναβάθμιση του σε Premium χρήστη, μπορεί να διακόψει τη συνδρομή του και να ξανα γίνει απλός χρήστης. Προσφέρουμε επιπλέον τη δυνατότητα στον χρήστη να δοκιμάσει την Premium συνδρομή δωρεάν για έναν μήνα χωρίς καμία χρηματική επιβάρυνση ή δέσμευση.
Επίσης η εφαρμογή αυτή παρέχει ασφαλές περιβάλλον για τους νεότερους χρήστες, αφού γίνεται αναφορά και αποκλεισμός χρηστών που κοινοποιούν ακατάλληλο περιεχόμενο από τους διαχειριστές. Αυτοί αφαιρούν το εν λόγω περιεχόμενο από την εφαρμογή και διαχειρίζονται τους διάφορους καλλιτέχνες, ελέγχοντας τους για όποιο ιστορικό παραβάσεων έχουν στην εφαρμογή. Έαν υπάρχει οποιοδήποτε θέμα, έχουν την ευελιξία είτε να δώσουν μια παρατήρηση (ονομαζόμενη "strike") είτε να αποκλείσουν άμεσα τους παραβάτες.
