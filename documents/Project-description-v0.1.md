# Αρχική Περιγραφή του Έργου
Σκοπεύουμε να υλοποιήσουμε μια εφαρμογή αναπαραγωγής μουσικής. Ο χρήστης έχει την δυνατότητα να ακούει μουσική, είτε αναζητώντας συγκεκριμένο περιεχόμενο είτε επιλέγοντας προτάσεις από την εφαρμογή, να σώζει και να κατεβάζει τα αγαπημένα του τραγούδια, να ακολουθεί καλλιτέχνες, να ενημερώνεται για επικείμενες συναυλίες, να δημιουργεί δημόσιες ή ιδιωτικές λίστες αναπαραγωγής και να αγοράζει περιεχόμενο από καλλιτέχνες.

Επιπλέον, ο χρήστης μπορεί να αιτηθεί να αναβαθμίσει τον λογαριασμό του στον λογαριασμό καλλιτέχνη, με τον οποίο μπορεί να κοινοποιεί νέα τραγούδια, να δημιουργεί άλμπουμ, να ενημερώνει το κοινό του για νέες συναυλίες και να προπωλεί την μουσική του.

Η χρήση της εφαρμογής αυτής είναι αποκλειστικά δωρεάν. Σκοπός της είναι να μιμιθεί τις πιο διάσημες εφαρμογές αναπαραγωγής τραγουδιών, χωρίς να επιβαρύνει τον χρήστη με επιπλέον χρηματικό κόστος ή διαφημήσεις που διακόπτουν την ακρόαση. Επίσης η εφαρμογή αυτή παρέχει ασφαλές περιβάλλον για τους νεότερους χρήστες, αφού γίνεται αναφορά και αποκλεισμός χρηστών που κοινοποιούν ακατάλληλο περιεχόμενο από τους διαχειριστές, οι οποίοι αφαιρούν το εν λόγω περιεχόμενο από την εφαρμογή.
